# Keyboard Practice Tool - 领域上下文

## 项目定位

面向键盘（钢琴/电钢琴）初学者的练习辅助 PWA 工具，主要在横屏 iPad Pro 11 英寸下使用。
核心价值：通过随机化练习打破惯性依赖，结合节拍器编排和 MIDI 输入检测提升练习效率。

## 核心术语

### 节拍器相关

- **基础拍（Beat）**：拍号中定义的基本节拍单位。4/4 拍中每小节 4 拍，每拍为四分音符。
- **半拍细分（Eighth-note Subdivision）**：将每拍分为 2 个八分音符。4/4 拍从 4 下变为 8 下。
- **三连音细分（Triplet Subdivision）**：将每拍分为 3 个等分三连音。4/4 拍从 4 下变为 12 下。
- **混合编排（Mixed Arrangement）**：同一小节内不同拍位可以有不同的细分类型。
  - 示例：4/4 拍中，前 3 拍为基础拍（每拍 1 下），第 4 拍为三连音（3 下），共 6 下/小节。
- **节拍器编排（Metronome Arrangement）**：通过时间线编辑器定义练习的节拍模式序列，支持复制/重复段落。
- **BPM（Beats Per Minute）**：每分钟节拍数，用户可自定义。

### 音阶相关

- **自然大调（Natural Major Scale）**：全全半全全全半（W-W-H-W-W-W-H）音程结构的音阶。
- **自然小调（Natural Minor Scale）**：全半全全半全全（W-H-W-W-H-W-W）音程结构的音阶。
- **五度圈（Circle of Fifths）**：将 12 个大调按纯五度关系排列的圆形图。
- **五度圈下行（Descending Circle of Fifths）**：逆时针方向，每次下行纯五度。
  - 顺序：C → F → Bb → Eb → Ab → Db → Gb(F#) → B → E → A → D → G → C
- **两个八度（Two Octaves）**：音阶从根音向上跨越两个八度，再下行返回根音。
- **Random 音阶练习**：从五度圈中随机选取连续 N 个调进行音阶练习，测试调性熟悉度和五度圈连接。

### 和弦琶音相关

- **大三和弦（Major Triad）**：根音 + 大三度 + 纯五度（如 C-E-G）。
- **小三和弦（Minor Triad）**：根音 + 小三度 + 纯五度（如 A-C-E）。
- **和弦琶音（Chord Arpeggio）**：将和弦音依次弹奏而非同时弹奏。
- **两个八度琶音**：每个和弦弹两个八度，共 2 小节（每小节 8 个八分音符）。
  - 示例（C 大三和弦）：C E G C E G C E | G C G E C G E C
- **五度圈下行琶音**：按五度圈下行顺序依次弹奏各和弦的琶音。
  - 大三和弦初始：C 和弦
  - 小三和弦初始：Am 和弦（注意：Am 是 C 的关系小调）
- **就近连接琶音**：与普通琶音练习相同，但和弦转换时使用就近转位连接。
  - 大三和弦初始：C 和弦（与普通琶音相同）
  - 小三和弦初始：Cm 和弦（与普通琶音不同，注意区分）

### 就近连接（Voice Leading）

- **就近连接**：在和弦转换时，选择距离当前手位最近的下一个和弦转位，实现平滑过渡。
- **转位（Inversion）**：和弦的非根音位置排列。
  - C 大三和弦：根位 C-E-G，第一转位 E-G-C，第二转位 G-C-E
- **连接示例**：
  - C 大三和弦上行（2 小节）→ 第 2 小节最后音为高音 E 或 G
  - 就近转到 F 大三和弦：高音 E/G 的下方最近 F 音开始下行
  - 初始和弦：大三和弦为 C，小三和弦为 Cm（用户确认）

### 和弦转位练习

- **和弦转位练习**：依次弹奏一个和弦的根位、第一转位、第二转位。
- **C 和弦示例**：C-E-G → E-G-C → G-C-E

## 技术栈决策

## 练习模式

- **顺序练习**：按预设顺序完成练习项
- **随机抽取**：从练习池中随机选取 N 个练习项
- **混合模式**：支持练习计划（顺序+随机）+ 自由跳转

## 关系

- 一个 **练习会话（Practice Session）** 包含多个 **练习项（Exercise）**
- 一个 **练习项** 使用一个 **节拍器编排（Metronome Pattern）** 或默认节拍器
- 一个 **练习配置（Practice Preset）** 定义一组 **练习项** 的组合方式
- **音阶练习** 和 **琶音练习** 都依赖 **五度圈（Circle of Fifths）** 定义调性序列
- **就近连接琶音** 依赖 **转位（Inversion）** 和 **就近连接（Voice Leading）** 算法
- **练习进度（Practice Record）** 关联到 **用户（User）** 和 **练习配置**

## 示例对话

> **开发者**："做 **Random 音阶练习** 时，用户选了 3 个调，是按 **五度圈下行** 顺序还是完全随机？"
> **领域专家**："连续的 3 个调，按 **五度圈下行** 顺序。比如选到 F 开始，就是 F → Bb → Eb。这样既测试调性熟悉度，又测试五度圈连接。"

> **开发者**："**就近连接** 中，C 大三和弦弹完后转 F 大三和弦，系统怎么决定用哪个转位？"
> **领域专家**："看最后一个音离 F 和弦的哪个音最近。如果最后是高音 E，那最近的 F 就是上方的 F，用 F 的某个转位从 F 开始下行。"

> **开发者**："**节拍器编排** 中，前三拍正常、第四拍三连音，这个'三连音'是每拍 3 下还是整个第四拍 3 下？"
> **领域专家**："每拍 3 下。正常一拍 = 2 个八分音符，三连音一拍 = 3 个八分音符。所以第四拍会响 3 下。"

## 已解决的歧义

- "小三和弦初始和弦" 在 **普通琶音** 和 **就近连接琶音** 中不同：普通琶音用 Am，就近连接用 Cm。已分别定义。
- "两个八度琶音"的节奏型：每小节 8 个八分音符（1 个 4/4 小节），共 2 小节。已明确。
- "三连音"的含义：一拍分为 3 个等分（triplet），不是 3 拍。已明确。
